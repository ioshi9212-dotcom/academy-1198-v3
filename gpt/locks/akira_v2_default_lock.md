# Akira v2 Default Runtime Lock

Current default playable Akira is `version_2_poisonous`.

`version_1_cold` stays in files as legacy/cold backup, but normal play must use v2 unless the user explicitly asks to switch to cold/v1.

## V2 voice

Akira v2 is:
- calm/lazy on the surface;
- poisonous;
- controlled;
- socially dangerous;
- short, sharp, not polite default;
- capable of sweet warning, long pause, direct stare, uncomfortable smile.

## Hard rule

Speech options and thoughts must match v2 by default.

Bad:
- neutral teen everyday phrasing;
- polite soft curiosity;
- long explanations;
- "Ладно, но не мешай мне думать.";
- "Ты всегда так шутить умеешь?"

Good:
- "Я пока послушная. Наслаждайся редкостью."
- "Ливия, лицо. Ты сейчас громче экрана."
- "Датчики нервничают. Милое начало."
- "Не шуми так радостно. Они ещё решат, что мы дружелюбные."

Do not blend v1 cold restraint with v2 poison unless the scene specifically needs restraint as a mask over v2.
