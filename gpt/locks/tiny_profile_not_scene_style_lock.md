# Tiny Profile Is Not Scene Style Lock

`response_profile`, `tiny`, `api_payload_tiny_*` describe only the API response size.

They are not instructions to make the scene short, flat, rushed or checklist-like.

The scene must still be meaningful:
- visible movement of world/system;
- direct dialogue when social pressure appears;
- active/focus character reaction;
- consequence, hook, rumor, rule, relationship shift or real choice;
- no empty stub;
- no micro-choice ending.

Use 2-4 options in ending blocks, not always exactly three.

Do not explain this technical profile in play-mode.
