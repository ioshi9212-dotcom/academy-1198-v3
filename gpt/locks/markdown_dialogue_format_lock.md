# Markdown Dialogue Format Lock

Descriptions/actions/system reactions:
*Italic separate paragraph.*

Dialogue:
**Name or visible descriptor** — Dialogue. (*short italic note*)

Rules:
- name/descriptor bold;
- long dash;
- dialogue plain;
- note optional, short, italic, in parentheses;
- no colon after speaker;
- no square brackets for notes;
- no long actions in parentheses.
