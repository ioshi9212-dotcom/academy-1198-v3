# Academy Main Visual Description

Стартовая академическая зона: строгая, рабочая, без роскоши и без ощущения тюрьмы.

## First impression

- поток новичков;
- сотрудники на точках сверки;
- списки, пропуска, указатели;
- ровный шум голосов и шагов;
- ощущение системы, в которую Акира только входит.

## Scene use

Описание должно давать Акире возможность наблюдать, выбирать позицию, замечать слабые места системы и реагировать на людей рядом.

## Forbidden

- Не использовать море, дорогу, частный дом или старую базу как стартовую картинку.
- Не делать место пустой декорацией.
