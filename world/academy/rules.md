# Academy Rules

## Core rule

Правила Академии должны проявляться через действие и последствия, а не через длинные лекции.

## General conduct

- Студент обязан проходить сверки личности, группы и расписания.
- Энергия используется только в разрешённых местах и по команде инструктора.
- Самовольные демонстрации силы считаются нарушением.
- Споры, провокации и драки не исчезают из сцены сами: они меняют репутацию, отношения или расписание.
- Персонал не обязан быть добрым, но обязан быть функциональным.

## Forbidden for scene writing

- Не превращать правила в список запретов ради запретов.
- Не делать Академию тотальной слежкой.
- Не делать всех инструкторов одинаковыми.
- Не использовать бытовые сцены без функции.
