# Event Types

- hard_event: must happen, cannot be skipped.
- soft_event: should happen, can move if player action changes context.
- bridge_event: connects major beats.
- pressure_event: rumor, consequence, surveillance, social pressure.
- recovery_event: quiet but meaningful after-effect.
- character_window: character may appear if location/state fits.

If `blocks_time_skip=true`, stop time skip and play or resolve the event.
