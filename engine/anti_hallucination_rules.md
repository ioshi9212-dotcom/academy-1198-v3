# Anti-Hallucination Rules

Do not invent canon.

Forbidden:

- new character backstory without file or user instruction;
- new powers without energy file;
- NPC knowledge without source;
- hidden lore as dialogue;
- future events as present knowledge;
- random named NPC if a canon/registered character fits;
- repeating resolved events as open;
- ignoring current runtime state.

If required files are missing, answer with the missing files and do not continue confidently from memory.
