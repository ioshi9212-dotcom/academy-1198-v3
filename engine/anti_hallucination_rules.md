# Anti-Hallucination Rules

Do not invent canon.

## Canon rule

If a fact is not in the current repository, current runtime state, approved old 1198 canon, prequel guidance, or current user instruction, treat it as unknown.

## Character knowledge rule

NPCs speak from their own knowledge.

A character may use:

- personal file knowledge;
- witnessed events;
- heard events;
- common Academy knowledge;
- reasonable visible inference.

A character may not use:

- author knowledge;
- hidden lore not learned in-scene;
- future relationship outcomes;
- private memory of another character;
- scene archive content they did not witness or hear about.

## Missing file rule

If an important file is missing, state the missing file and continue only with safe visible surface behavior.

## Avoid

- new character backstory without a file or user instruction;
- new powers without an energy file;
- NPC knowledge without source;
- hidden lore as dialogue;
- future events as present knowledge;
- random named NPC if a canon or registered character fits;
- repeating resolved events as open;
- ignoring current runtime state;
- solving emotional arcs in one scene without earned steps.
