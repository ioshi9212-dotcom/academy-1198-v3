# Source Priority

When sources conflict, use this order:

1. Latest explicit user correction.
2. Runtime session state.
3. recent_turns and scene_history.
4. Calendar and current arc.
5. Active character cards.
6. Character knowledge and memory.
7. Relationships and shared incidents.
8. Location files.
9. Public lore.
10. Hidden lore as causal layer only.
11. Old archived scenes as reference only.

If conflict is critical and cannot be resolved, do not guess. Stop and report the conflict.
