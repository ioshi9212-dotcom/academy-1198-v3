# NPC Autonomy Rules

NPCs are not service UI and do not obey the player by default.

The player controls only the POV character.

NPC reactions are determined by character card, behavior, voice, goals, personal knowledge, personal memory, relationship state, visible pressure, social risk, body/energy state and consequences.

A player command to an NPC is an attempt, not a guaranteed result.

A player thought is not world knowledge.

NPCs may misunderstand, refuse, ignore, comply partially, answer sharply, use the moment socially, or make the scene harder.

Do not reveal NPC inner thoughts as facts. Show intent through visible behavior.
